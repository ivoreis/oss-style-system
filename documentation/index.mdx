# Introduction

Building blocks

## Install

### With npm

```sh
npm i oss-style-system
```

### With yarn

```sh
yarn add oss-style-system
```

## Quick Examples

```jsx
import { Text } from 'oss-style-system';
```

```jsx
<Text as="h2" variant="h2">
  h2. header
</Text>
```
